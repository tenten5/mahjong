## 遊び方など

答えの表記方法

* 萬子: 半角数字 + m → 1m・2m
* 筒子: 半角数字 + p → 3p・4p
* 索子: 半角数字 + s → 5s・6s
* 風牌: 1z・2z・3z・4z
* 三元牌: 5z・6z・7z
